flip_twice <- function(pr = 0.5) {
  sample(c("H", "T"), 
         replace = T, 
         prob = c(pr, 1 - pr)) %>% 
    paste0(collapse = "")
}

tibble(flips = replicate(500, flip_twice(0.7)),
       keep = flips %in% c("HT", "TH")) %>% 
  ggplot(aes(flips, fill = keep)) +
  geom_bar() +
  coord_flip() +
  theme_minimal() +
  scale_fill_brewer()

What‚Äôs neat is that we can just keep expanding this. You‚Äôve got a biased die? Just roll six times. Rolling a six-sided die six times actually has quite a few possible outcomes, so maybe let‚Äôs try a three sided one?

roll_three <- function(pr = c(1/6, 1/6, 1/6)) {
  sample(1:3,
         replace = T,
         prob = pr) %>%
    .[1:3] %>% 
    paste0(collapse = "")
}

tibble(
  rolls = replicate(100000, roll_three(c(0.2, 0.2, 0.6)))
) %>% 
  ggplot(aes(rolls)) +
  geom_bar() +
  coord_flip() +
  theme_minimal() +
  scale_fill_brewer()

You‚Äôll notice that there are of course more than three equally likely possibilities now, but choose three to keep and you‚Äôve got a fair die. There may be a more clever way to extend this, let me know!
